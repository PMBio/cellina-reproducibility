# cellina-reproducibility
Repository containing all scripts and notebooks to reproduce results for `cellina`. 

## Sections
1. Ablation study of each component of `cellina` - classifier, discriminator and edge loss
2. Marginal log likelihood for unseen region/cell type combination
3. Benchmark measuring disentanglement on 3 datasets
4. Counterfactual prediction of cell states in unseen niche

## Datasets
1. Simulation data with left and right ventricles and a stressed state
2. Melanoma (Cosmx Nanostring) [source](https://www.nature.com/articles/s41467-025-58089-7)
3. Lung NSCLC (Cosmx Nanostring) [source](https://nanostring.com/products/cosmx-spatial-molecular-imager/ffpe-dataset/nsclc-ffpe-dataset/)
4. Colorectal cancer CRC (WT Cosmx Nanostring) [source](https://www.biorxiv.org/content/10.1101/2025.06.23.660674v1)
